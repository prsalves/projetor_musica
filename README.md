# 🎯 Projetor de música

Este simples projeto em python exibe a letra e reproduz a música através da biblioteca Pygame.

## 🚀 Funcionalidades

* Exibe a letra da música de acordo com o tempo dela, sincronizando com o som
* Possui um terminal colorido e estilizado
* 
## !! Problemas de funcionalidade !!

* Exige um arquivo externo para reproduzir a música
* Termina de forma abrupta, e da um erro no final
* Exige que a letra seja digitada no formato de dicionário, sendo um trabalho muito trabalhoso colocar a letra e o tempo certinho para fazer a sincronização

## 🛠️ Tecnologias utilizadas

* Linguagem de programação Python
* Biblioteca colorama para estilizar o terminal
* Biblioteca Pygame para reproduzir a música
* Arquivo de música separado para tocar a música

## 🎮 Como utilizar

1. Adicione o arquivo da música na mesma pasta que o arquivo python.
2. Instale a biblioteca colorama e pygame, com pip install colorama e pip install pygame
3. Execute o arquivo python
4. Aproveite a música

## 📚 Objetivo do projeto

Esse projeto foi desenvolvido com o objetivo de ser um protótipo de uma espécie de Spotify pessoal, tendo como funcionalidades:

* Tocar músicas offline
* Exibir a letra das músicas
* Servir apenas para uso pessoal, sem objetivos comerciais
